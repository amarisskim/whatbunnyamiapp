# whatbunnyamiapp

A personality and personal taste quiz app with a social element. Take a quiz to discover what kind of bunny you are, share your results, and compare with friends.

## About

whatbunnyamiapp is a mobile-first web app built for fun and social sharing. Users answer personality and taste questions, get a bunny-themed result, and can share or compare results with friends.

## Tech Stack

- **Framework:** Next.js (React)
- - **Styling:** Tailwind CSS
  - - **Database & Auth:** Supabase
    - - **Hosting:** Vercel
     
      - ## Getting Started
     
      - First, install dependencies:
     
      - ```bash
        npm install
        ```

        Then run the development server:

        ```bash
        npm run dev
        ```

        Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.
